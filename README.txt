A library for fitting self-organising maps[1] on toroidal topology
written by Simon Wilshin, swilshin@gmail.com, Jul 2018.

This work was completed with support from the Royal Veterinary College, 
London (www.rvc.ac.uk).

Full documentation is provided by epydoc, a script for producing this 
documentation on linux systems is provided (epydoc.sh). Example fits a 
simple self-organising map to some data on a 2-torus.

[1] Kohonen, Teuvo. "Self-organized formation of topologically correct feature 
  maps." Biological cybernetics 43.1 (1982): 59-69.
